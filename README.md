## 🚀 Getting Started

To compile and run any file:

```bash
g++ -std=c++17 -o output filename.cpp
./output
```

Or with warnings enabled (recommended):

```bash
g++ -std=c++17 -Wall -Wextra -o output filename.cpp
```
